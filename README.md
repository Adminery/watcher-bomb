## ⚡ WHAT IS WATCHER BOMB?

**Watcher Bomb** is a professional SMS bombing tool with **29 active services** (Snapp, Tapsi, Divar, Digikala, etc.).  
It features a **hacker-style UI**, **real-time statistics**, and **multi-threading** support.

> ⚠️ **FOR EDUCATIONAL PURPOSES ONLY!** Use only on your own numbers.

## 📥 INSTALLATION

### 🟢 Termux (Android)
```bash
pkg update && pkg upgrade
pkg install python3 git
git clone https://github.com/Adminery/watcher-bomb.git
cd watcher-bomb
bash install.sh
python3 watcher.py
```

🔴 Kali Linux / Ubuntu / Debian

```bash
sudo apt update
sudo apt install python3 python3-pip git
git clone https://github.com/Adminery/watcher-bomb.git
cd watcher-bomb
bash install.sh
python3 watcher.py
```

🟡 Arch Linux / Manjaro

```bash
sudo pacman -S python python-pip git
git clone https://github.com/Adminery/watcher-bomb.git
cd watcher-bomb
bash install.sh
python3 watcher.py
```

🔵 Windows (PowerShell as Admin)

```bash
git clone https://github.com/Adminery/watcher-bomb.git
cd watcher-bomb
pip install -r requirements.txt
python watcher.py
```

---

🎮 USAGE

```
python3 watcher.py
```

Then follow the prompts:

1. Enter target number (with +98 or 09xx)
2. Set threads (1-20, default 5)
3. Set cycles (default 1)
4. Set delay between cycles (default 3s)
